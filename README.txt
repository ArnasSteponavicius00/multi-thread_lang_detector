Multi-Threaded Language Detector.
==================================================================
Arnas Steponavicius - G00361891 - Object Oriented Programming.

An Application that takes in a Subject/Dataset file of all languages and a Query file.
Then outputs the most likely language.

INCLUDED IN FOLDER:
==============================================
-docs folder with the documentation of the program.
-design.png, an image file of the UML diagram of the program.

FEATURES:
==============================================
- Inputs Subject/Dataset file location.
- Specify the amount of kmers to use. (higher kmer count means more accurate results, at the loss of higher resource usage.)
- Input Query file location you wish to find out the language of.

HOW TO USE:
==============================================
- Uzip folder anywhere.
- run the following command
--------------------------------------------
java -cp oop.jar ie.gmit.sw.Runner
--------------------------------------------
